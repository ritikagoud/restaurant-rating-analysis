# Restaurant Rating Analysis (India)

## Overview
This project performs exploratory data analysis on Indian restaurant ratings to understand how pricing, service features, and location influence customer satisfaction. The analysis is done entirely using Python.

## Dataset
- Source: Kaggle (Indian Restaurants Dataset)
- Size: 200,000+ restaurant records
- Note: Raw dataset is not uploaded due to size constraints.

## Objectives
- Analyze the relationship between price range and restaurant ratings
- Study the impact of online ordering on ratings
- Examine how table reservation availability affects customer satisfaction
- Compare average ratings across major Indian cities

## Key Insights
- Average restaurant ratings increase with price range but plateau after the ₹2000–5000 range.
- Restaurants with table reservation facilities receive significantly higher ratings.
- Online ordering shows a moderate positive impact on ratings.
- Major cities show relatively consistent rating levels with small regional variations.

- pandas
- numpy
